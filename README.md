ATM Simulation in Python:
    This project is a menu-driven ATM simulation program written in Python.It allows users to authenticate using an account number and PIN,
    then perform common ATM operations such as deposit, withdrawal, balance enquiry, PIN change, and mini-statement.

This project is ideal for beginners learning:
    - Python loops
    - Conditionals
    - Dictionaries
    - Menu-driven applications
    -Basic banking logic

Features:
  User Authentication
    - Account number verification
    - PIN validation
    - 3 failed attempts → account locked

 Banking Operations
    - Deposit money
    - Withdraw money with balance check
    - Check current balance

PIN Management
    - Change current PIN with confirmation

Mini Statement
    - Shows last few transactions
    - Records deposits & withdrawals

Transaction Logging
    - Stores each successful transaction
    - Displays in reverse chronological order

Session/Exit Handling
    - Clean menu-driven interface
    - “Thank you” message on exit

Make sure Python is installed:
    - python --version

Run the ATM program:
    - python atm.py
